# 🚀 Node.js Project Base (Backend Foundation)

Bu proje, Node.js ve Express.js kullanarak modern web uygulamalarının arka plan (backend) mimarisini, sunucu yönetimini ve API mantığını sıfırdan anlamak ve uygulamak amacıyla geliştirilmiş temel bir backend projesidir.

## 🎯 Projenin Amacı ve Mantığı

Projenin temel odağı, hazır kütüphanelerin arkasında dönen mantığı kavramaktır. Bu kapsamda şu backend temelleri üzerine inşa edilmiştir:
* **HTTP Sunucu Yönetimi:** Node.js tabanlı yerel bir sunucu (`localhost`) ayağa kaldırma ve yönetme.
* **Routing (Yönlendirme):** Express.js kullanarak istemciden (client) gelen istekleri (GET/POST) karşılama ve doğru verilere yönlendirme.
* **Middleware ve Entegrasyon:** Sunucu döngüsünü kontrol etme ve `nodemon` gibi araçlarla geliştirme sürecini otomatikleştirme.

## ⏳ Mevcut Durum: Planlı Ara (On Hold)

**Bu projeye şu anda geçici bir ara verilmiştir.** 

Backend mimarisini çok daha efektif yönetebilmek, asenkron yapıları (Async/Await) ve veri manipülasyonunu tam anlamıyla profesyonel seviyeye taşımak adına, şu aşamada **JavaScript (Vanilla JS) ve Frontend** teknolojilerindeki yetkinliklerimi en üst seviyeye çıkarmaya odaklandım. 

JavaScript ekosisteminde tamamen uzmanlaştıktan sonra bu projeye geri dönülecek ve şu özellikler entegre edilecektir:
* Gelişmiş RESTful API mimarisi,
* Veritabanı (Database) bağlantıları,
* Kullanıcı yetkilendirme (Auth) sistemleri.

## 🛠️ Başlangıç Teknolojileri

* **Runtime:** Node.js
* **Framework:** Express.js
* **Development Tool:** Nodemon
